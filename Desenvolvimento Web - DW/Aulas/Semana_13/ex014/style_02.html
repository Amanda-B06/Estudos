CSS Interno

Um estilo interno é aquele que fica configurado dentro da área do <head> do seu documento.

A grande vantagem dessa maneira em relação o modelo inline é que podemos escrever uma configuração única para todos os elementos da mesma família. Por exemplo, todos os parágrafos deste documento estão com a mesma configuração de cor, e eu só preciso escrever isso uma vez.

A grande desvantagem deste modelo em relação a ultima estrutura que veremos é que no CSS externo (a estrutura do próximo exercício) é o que estilo não pode ser reaproveitado por outras páginas.

IMPOTANTE: Você pode misturar CSS inline com CSS interno em um mesmo documento sem problemas! Vamos fazer um pequeno TESTE! É deste processo que aplicamos os estilos em diferentes níveis que surge o conceito de "Folhas de Estilo em Cascata".
